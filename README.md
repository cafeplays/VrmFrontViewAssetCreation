# VrmFrontViewPropMaker

Welcome to the VrmFrontViewPropMaker project! This project is designed for creating, configuring, and managing props for VrmFrontView.

## Recommended External Assets

To get the most out of your props and enhance their visual quality and behavior, we highly recommend adding the following assets to your Unity project:

- **[lilToon](https://lilxyzw.github.io/lilToon/#/)**: A highly versatile and lightweight shader, perfect for anime-style characters and stylised props.
- **[Poiyomi Shaders](https://www.poiyomi.com/)**: A feature-rich rendering shader widely used in the community, offering extensive customization options.
- **[AudioLink](https://github.com/llealloo/vrc-udon-audio-link)**: A system that allows shaders and materials to react to audio in real-time, great for making your props reactive to music or sound effects.
- **[Magica Cloth 2](https://assetstore.unity.com/packages/tools/physics/magica-cloth-2-258500)**: A powerful, high-performance physics engine for realistic cloth, hair, and accessory simulation.

## Getting Started

1. Open this project in Unity.
2. If you choose to use the recommended assets above, import their respective `.unitypackage` files or add them via the Unity Package Manager.
3. Start creating and configuring your props!
